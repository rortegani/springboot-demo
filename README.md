## Spring Boot Demo  
CRUD de Productos y Categorías con Spring Boot, Spring Data JPA y MySQL

## Descripción 
Este proyecto implementa una API REST para la gestión de categorías y productos, aplicando arquitectura en capas con entidades, repositorios, servicios y controladores.  
El sistema permite operaciones CRUD y búsquedas personalizadas, con persistencia automática en base de datos MySQL.

## Tecnologías 
- Java 21
- Spring Boot 3.5.7
- Spring Web
- Spring Data JPA
- MySQL Driver
- Springdoc OpenAPI (Swagger UI)
- Maven
- NetBeans IDE
